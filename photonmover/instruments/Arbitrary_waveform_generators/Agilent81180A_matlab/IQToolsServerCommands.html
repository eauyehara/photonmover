IQTools Server Command Set

IQTools Server 1.0

Generic Commands (*)

Command: *IDN?
Type: Query Only
Description: Returns the name and version of the server
Example: *IDN?

Command: *OPC?
Type: Query Only
Description: Returns a '1' when all operations are complete
Example: *OPC?

Command: *RST
Type: Send Only
Description: Resets the server to its default values
Example: *RST

Command: *CLS
Type: Send Only
Description: Clears the error queue
Example: *CLS

Instrument Commands (:INST)

Command: :INST:MODE <Mode String>
Type: Send and Query
Description: Gets or sets the server's instrument mode (N5194A_250MHz, N5194A_2GHz, M8190A_12bit, M8190A_14bit)
Example: :INST:MODE?; :INST:MODE N5194A_250MHz

Command: :INST:SCPI <SCPI Command String>
Type: Send Only
Description: Sends a SCPI command to the configured instrument
Example: :INST:SCPI *RST

Command: :INST:CHMAP <Channel Map String>
Type: Send and Query
Description: Gets or sets the configured instrument's channel mapping
Example: :INST:CHMAP?; :INST:CHMAP [1 0; 0 1]

Command: :INST:SNUM <segmentNumber>
Type: Send and Query
Description: Gets or sets the segment number into which a waveform is downloaded
Example: :INST:SNUM?; :INST:SNUM 1

Command: :INST:SNAME <segmentName>
Type: Send and Query
Description: Gets or sets the segment name that is used for download
Example: :INST:SNAME?; :INST:SNAME 'waveform001'

Command: :INST:FTYPE <Channel Map String>
Type: Send and Query
Description: Gets or sets the server's file save type
Example: :INST:FTYPE?; :INST:FTYPE '16-bit I/Q MSB'

Command: :INST:CORR <0 or 1>
Type: Send and Query
Description: Gets or sets the if corrections are enabled
Example: :INST:CORR?; :INST:CORR 1

Command: :INST:FCORR <File Path>
Type: Send and Query
Description: Gets or sets the correction file path, which overwrites the current file present
Example: :INST:FCORR?; :INST:FCORR 'ampCorr.mat'

Command: :INST:RFFREQ <Frequency in Hz>
Type: Send and Query
Description: Gets or sets the carrier frequency for the signal if using a signal generator
Example: :INST:RFFREQ?; :INST:RFFREQ 4E9

Command: :INST:RFPOW <Power in dBm>
Type: Send and Query
Description: Gets or sets the carrier power for the signal if using a signal generator
Example: :INST:RFPOW?; :INST:RFPOW -20

Command: :INST:RFON <0 or 1>
Type: Send and Query
Description: Gets or sets the carrier output state
Example: :INST:RFON?; :INST:RFON 1

Calibration Commands (:CAL)

Command: :CAL:TYPE <Calibration Type>
Type: Send and Query
Description: Gets or sets the current calibration type to perform
Example: :CAL:TYPE?; :CAL:TYPE vsachannel

Command: :CAL:FSPAN <Span in Hz>
Type: Send and Query
Description: Gets or sets the current calibration's frequency span in Hz
Example: :CAL:FSPAN?; :CAL:FSPAN 1E9

Command: :CAL:FSPAC <Spacing in Hz>
Type: Send and Query
Description: Gets or sets the current calibration's frequency spacing in Hz
Example: :CAL:FSPAC?; :CAL:FSPAC 1E6

Command: :CAL:FOFFS <Offset in Hz>
Type: Send and Query
Description: Gets or sets the current calibration's frequency offset in Hz (useful to avoid LO feedthru)
Example: :CAL:FOFFS?; :CAL:FOFFS 10E3

Command: :CAL:FCENT <Tune Frequency in Hz>
Type: Send and Query
Description: Gets or sets the current calibration's center frequency in Hz
Example: :CAL:FCENT?; :CAL:FCENT 10E9

Command: :CAL:TSETT <Cal Measurement Time in S>
Type: Send and Query
Description: Gets or sets the current calibration's measurement time in S
Example: :CAL:TSETT?; :CAL:TSETT 2.5

Command: :CAL:FSAV <Calibration file save path>
Type: Send and Query
Description: Gets or sets the file path a calibration file will be saved to
Example: :CAL:FSAV?; :CAL:FSAV Cal_Save_FC_1E9

Command: :CAL:INIT <0 or 1>
Type: Send and Query
Description: Gets or sets whether or not the calibration runs through an initialization
Example: :CAL:INIT?; :CAL:INIT 1

Command: :CAL:ARANG <0 or 1>
Type: Send and Query
Description: Gets or sets whether or not the calibration autoranges the front end
Example: :CAL:ARANG?; :CAL:ARANG 1

Command: :CAL:RANG <Power in dBm>
Type: Send and Query
Description: Gets or sets the power the calibration will range to
Example: :CAL:RANG?; :CAL:RANG -10

Command: :CAL:EXE
Type: Send Only
Description: Performs a calibration based on the parameters defined
Example: :CAL:EXE

Command: :CAL:SAVE
Type: Send Only
Description: Saves the current calibration to the defined path
Example: :GEN:PULS:SAVE

System Commands (:SYST)

Command: :SYST:TCPIP:CLOS
Type: Send Only
Description: Closes the current TCPIP Server connection and listens again for a new connection
Example: :SYST:TCPIP:CLOS

Command: :SYST:ERR?
Type: Query Only
Description: Return the last error in the error queue or '0' if no errors are present
Example: :SYST:ERR?

Command: :SYST:DEBUG <0 or 1>
Type: Send and Query
Description: Gets or sets if debug mode is enabled
Example: :SYST:DEBUG?; :SYST:DEBUG 1

Script Commands (:SCRIPT)

Command: :SCRIPT:EVAL '<MATLAB syntax expression>'
Type: Send and Query
Description: Evaluates the expression and/or returns the expression's result
Example: :SCRIPT:EVAL? '1 + 1'; :SCRIPT:EVAL 'plot(1:10, 2:2:20);'

Command: :SCRIPT:COMMAND:SET '<MATLAB syntax expression>'
Type: Send and Query
Description: Gets or sets the command to evaulate with :EXE or add to a list with :LIST:ADD
Example: :SCRIPT:COMMAND:SET?; :SCRIPT:COMMAND:SET 'plot(1:10, 2:2:20);'

Command: :SCRIPT:COMMAND:RESET
Type: Send Only
Description: Resets the current set command
Example: :SCRIPT:COMMAND:RESET

Command: :SCRIPT:COMMAND:EXE
Type: Send Only
Description: Executes the current set command
Example: :SCRIPT:COMMAND:EXE

Command: :SCRIPT:LIST:ADD
Type: Send Only
Description: Adds the current set command to the list of commands to execute with :LIST:EXE
Example: :SCRIPT:COMMAND:EXE

Command: :SCRIPT:LIST:RESET
Type: Send Only
Description: Resets the list of commands
Example: :SCRIPT:COMMAND:RESET

Command: :SCRIPT:LIST:COUNT
Type: Query Only
Description: Gets the number of commands currently in the command list
Example: :SCRIPT:COMMAND:LIST:COUNT?

Command: :SCRIPT:LIST:EXE
Type: Send Only
Description: Executes the current list of commands
Example: :SCRIPT:COMMAND:LIST:EXE

Pulse Generation Commands (:GEN:PULS)

Command: :GEN:PULS:SRAT <Sample Rate in Hz>
Type: Send and Query
Description: Gets or sets the current pulse configuration's sample rate in Hz
Example: :GEN:PULS:SRAT?; :GEN:PULS:SRAT 2E9

Command: :GEN:PULS:RESET
Type: Send Only
Description: Resets the current pulse configuration to its default values
Example: :GEN:PULS:RESET

Command: :GEN:PULS:DWEL <Dwell Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse configuration's dwell time/s (duration/s) in seconds
Example: :GEN:PULS:DWEL?; :GEN:PULS:DWEL 100E-6; :GEN:PULS:DWEL [100E-6, 10E-6]

Command: :GEN:PULS:WIDT <Dwell Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse configuration's width/s in seconds
Example: :GEN:PULS:WIDT?; :GEN:PULS:WIDT 10E-6; :GEN:PULS:WIDT [10E-6, 1E-6]

Command: :GEN:PULS:RISE <Rise Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse configuration's rise time/s in seconds
Example: :GEN:PULS:RISE?; :GEN:PULS:RISE 10E-9; :GEN:PULS:RISE [10E-9, 20E-9]

Command: :GEN:PULS:FALL <Fall Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse configuration's fall time/s in seconds
Example: :GEN:PULS:FALL?; :GEN:PULS:FALL 10E-9; :GEN:PULS:FALL [10E-9, 20E-9]

Command: :GEN:PULS:DELA <Delay Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse configuration's delay time/s in seconds
Example: :GEN:PULS:DELA?; :GEN:PULS:DELA 10E-9; :GEN:PULS:DELA [10E-9, 20E-9]

Command: :GEN:PULS:AMP <Relative Amplitudes in dB>
Type: Send and Query
Description: Gets or sets the current pulse configuration's relative amplitude/s in dB
Example: :GEN:PULS:AMP?; :GEN:PULS:AMP [0, -6]

Command: :GEN:PULS:PHAS <Phase in Degrees>
Type: Send and Query
Description: Gets or sets the current pulse configuration's phase/s in degrees
Example: :GEN:PULS:PHAS?; :GEN:PULS:PHAS 90; :GEN:PULS:PHAS [90, 45]

Command: :GEN:PULS:FREQOF <Frequency Offset in Hz>
Type: Send and Query
Description: Gets or sets the current pulse configuration's frequency offset/s in Hz
Example: :GEN:PULS:FREQOF?; :GEN:PULS:FREQOF 10E6; :GEN:PULS:FREQOF [10E6, -50E6]

Command: :GEN:PULS:SHAP <Pulse Shape String>
Type: Send and Query
Description: Gets or sets the current pulse configuration's pulse shape/s (Raised Cosine, Trapezodial, Gaussian, Zero signal during rise time)
Example: :GEN:PULS:SHAP?; :GEN:PULS:SHAP Raised Cosine

Command: :GEN:PULS:MOD <Modulation Type Cell String Array>
Type: Send and Query
Description: Gets or sets the current pulse configuration's modulation type/s (None, Increasing, Decreasing, V-Shape, Inverted V, Barker-2 +-, Barker-2 ++, Barker-3, Barker-4 ++-+, Barker-4 +++-, Barker-5, Barker-7, Barker-11, Barker-13, Frank-4, Frank-6, FMCW)
Example: :GEN:PULS:MOD?; :GEN:PULS:MOD {'Increasing'}, :GEN:PULS:MOD {'Increasing', 'None'}

Command: :GEN:PULS:FMOD <MATLAB Syntax Modulation Formula>
Type: Send and Query
Description: Gets or sets the current pulse configuration's FM modulation formula. Use the variable "x" in the formula, which varies from 0 to 1 throughout the pulse
Example: :GEN:PULS:FMOD?; :GEN:PULS:FMOD cos(pi*(x-1))

Command: :GEN:PULS:FSPA <Frequency Span in Hz>
Type: Send and Query
Description: Gets or sets the current pulse configuration's frequency FM span/s in Hz
Example: :GEN:PULS:FSPA?; :GEN:PULS:FSPA 10E6; :GEN:PULS:FSPA [10E6, 50E6]

Command: :GEN:PULS:PMOD <MATLAB Syntax Modulation Formula>
Type: Send and Query
Description: Gets or sets the current pulse configuration's PM modulation formula. Use the variable "x" in the formula, which varies from 0 to 1 throughout the pulse
Example: :GEN:PULS:PMOD?; :GEN:PULS:PMOD zeros(1,length(x)))

Command: :GEN:PULS:PTRAN <Phase Transition Type Cell String Array>
Type: Send and Query
Description: Gets or sets the current pulse configuration's phase transition type/s (Coherent, Continuous, Exact, Bump)
Example: :GEN:PULS:PTRAN?; :GEN:PULS:PTRAN {'Continuous'}; GEN:PULS:PTRAN {'Continuous', 'Coherent'}]

Command: :GEN:PULS:NORM <0|1>
Type: Send and Query
Description: 1: scale to full DAC range, 0: use :GEN:PULS:AMP as dbFS
Example: :GEN:PULS:NORM?; :GEN:PULS:NORM 0

Command: :GEN:PULS:FSAV <File save path>
Type: Send and Query
Description: Gets or sets the current pulse configuration's file save path
Example: :GEN:PULS:FSAV?; :GEN:PULS:FSAV Pulse_10_us

Command: :GEN:PULS:SAVE
Type: Send Only
Description: Calculates the I/Q Data for a pulse defined by its current settings, then saves to its defined path
Example: :GEN:PULS:SAVE

Command: :GEN:PULS:VSA
Type: Send Only
Description: Calculates the I/Q Data for a pulse defined by its current settings, then loads it to 89601B VSA to preview
Example: :GEN:PULS:VSA

Command: :GEN:PULS:DOWNLOAD
Type: Send Only
Description: Calculates the I/Q Data for a pulse defined by its current settings, then loads it to the defined instrument
Example: :GEN:PULS:DOWNLOAD

Pulse Combining Generation Commands (:GEN:PULS:COMB)

Command: :GEN:PULS:COMB:RESET
Type: Send Only
Description: Resets the current lists of pulses to combine
Example: :GEN:PULS:COMB:RESET

Command: :GEN:PULS:COMB:TSTART <Start Time in Seconds>
Type: Send and Query
Description: Gets or sets the current pulse to combine's start time in seconds
Example: :GEN:PULS:COMB:TSTART?; :GEN:PULS:COMB:TSTART 100E-6

Command: :GEN:PULS:COMB:AMP <Amplitude in dBm>
Type: Send and Query
Description: Gets or sets the current pulse to combine's amplitude in dBm
Example: :GEN:PULS:COMB:AMP?; :GEN:PULS:COMB:AMP -20

Command: :GEN:PULS:COMB:PTRAN <Phase Transition Type String>
Type: Send and Query
Description: Gets or sets the current pulse configuration's phase transition type/s (Coherent, Continuous, Exact, Bump)
Example: :GEN:PULS:COMB:PTRAN?; :GEN:PULS:COMB:PTRAN Coherent

Command: :GEN:PULS:COMB:PHAS <Phase Offset in Degrees>
Type: Send and Query
Description: Gets or sets the current pulse to combine's phase offset in degrees
Example: :GEN:PULS:COMB:PHAS?; :GEN:PULS:COMB:PHAS 90

Command: :GEN:PULS:COMB:ADD
Type: Send Only
Description: Adds the currently configured pulse combine entry to the list of pulses to combine
Example: :GEN:PULS:COMB:ADD

Command: :GEN:PULS:COMB:COUNT
Type: Query Only
Description: Gets the number of pulses that will be processed
Example: :GEN:PULS:COMB:COUNT?

Command: :GEN:PULS:COMB:OFFSETAMP
Type: Query Only
Description: Gets the amplitude offset of the calculated pulses (if the combined pulses constructively interfere, they may clip the DAC, so the the levels are normalized and managed with an offset)
Example: :GEN:PULS:COMB:OFFSETAMP?

Command: :GEN:PULS:COMB:OFFSETTIME
Type: Query Only
Description: Gets the time offset of the calculated pulses (to save samples, the first pulse can be offset in time to zero)
Example: :GEN:PULS:COMB:OFFSETTIME?

Command: :GEN:PULS:COMB:FSAV <File save path>
Type: Send and Query
Description: Gets or sets the current pulse combination configuration's file save path
Example: :GEN:PULS:COMB:FSAV?; :GEN:PULS:COMB:FSAV Pulse_Combine_100_us

Command: :GEN:PULS:COMB:SAVE
Type: Send Only
Description: Calculates the I/Q Data for a pulse combination defined by its current settings, then saves to its defined path
Example: :GEN:PULS:COMB:SAVE

Command: :GEN:PULS:COMB:VSA
Type: Send Only
Description: Calculates the I/Q Data for a pulse combination defined by its current settings, then loads it to 89601B VSA to preview
Example: :GEN:PULS:COMB:VSA

Command: :GEN:PULS:COMB:DOWNLOAD
Type: Send Only
Description: Calculates the I/Q Data for a pulse combination defined by its current settings, then loads it to the defined instrument
Example: :GEN:PULS:COMB:DOWNLOAD

Multi-tone Generation Commands (:GEN:TONE)

Command: :GEN:TONE:RESET
Type: Send Only
Description: Resets the current tone configuration to its default values
Example: :GEN:TONE:RESET

Command: :GEN:TONE:SRAT <Sample Rate in Hz>
Type: Send and Query
Description: Gets or sets the current tone configuration's sample rate in Hz.
Example: :GEN:TONE:SRAT?; :GEN:TONE:SRAT 2E9

Command: :GEN:TONE:FREQ <frequencies in Hz>
Type: Send and Query
Description: Gets or sets the current tone frequencies in Hz
Example: :GEN:TONE:FREQ?; :GEN:TONE:FREQ 100E6; :GEN:TONE:FREQ [100e6, 200e6, 500e6]

Command: :GEN:TONE:MAG <magnitudes in dBFS>
Type: Send and Query
Description: Gets or sets the current tone magnitude(s) in dBFS
Example: :GEN:TONE:MAG?; :GEN:TONE:MAG -12; :GEN:TONE:MAG [-10 -20 -30]

Command: :GEN:TONE:PHASE <list of phase values in degrees or 'Random'>
Type: Send and Query
Description: Gets or sets the current phase(s) in degrees
Example: :GEN:TONE:PHASE?; :GEN:TONE:PHASE 'Random'; :GEN:TONE:PHASE [0 90 180 270]

Command: :GEN:TONE:NORM <1|0>
Type: Send and Query
Description: When set to 1 (=default value), waveforms are scaled to use the full DAC range. When set to 0, scaling to full DAC range is only performed if the DAC range is exceeded.
Example: :GEN:TONE:MAG?; :GEN:TONE:MAG -12; :GEN:TONE:MAG [-10 -20 -30]

Command: :GEN:TONE:DOWNLOAD
Type: Send only
Description: Calculates the I/Q data for a multi-tone signal defined by its current settings, then loads it to the defined instrument
Example: :GEN:TONE:DOWNLOAD
